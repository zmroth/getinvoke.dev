System requirements — Windows installer

  • Operating system Windows 10 or 11 (64-bit)
  • RAM 4 GB minimum
  • GPU NVIDIA with CUDA 12.x (recommended)
  • CPU mode Works without GPU — just slower
  • Microphone Any input device
  • Disk space ~200 MB
  • Internet For license activation + AI reformatter

GPU transcription is sub-second. CPU mode takes a few seconds per dictation. Use CUDA Toolkit 12.x for GPU acceleration.

System requirements — CLI (pip install)

  • Python 3.11 or newer
  • Operating system Windows, Linux, or macOS
  • GPU NVIDIA CUDA (Windows/Linux) or CPU mode
  • Microphone Any input device
  • Disk space ~100 MB + Whisper model (~1.5 GB)

CPU mode works on any platform. macOS uses CPU transcription (Metal GPU support coming with the native app).

Installation — Windows installer

  1. Download the installer from GitHub Releases. Look for the .exe file.
  2. Run the installer. Windows may show a SmartScreen warning — click "More info" then "Run anyway." The app is code-signed but new to SmartScreen.
  3. Follow the setup wizard. Choose your audio device, pick a reformatter backend (OpenRouter, Claude CLI, Ollama, or none), and set your hotkey.
  4. Start dictating. Hold your hotkey, speak, release. Text appears at your cursor. Free for 7 days — no license key needed.

Installation — CLI (pip install)

  1. Install Python 3.11+ if you don't have it. python.org/downloads
  2. Install Invoke. Run pip install getinvoke or pipx install getinvoke for an isolated install.
  3. Run setup. dictate setup walks you through audio device, reformatter backend, and hotkey configuration.
  4. Start dictating. Run dictate run, hold your hotkey, speak, release. Free for 7 days — no license key needed.
